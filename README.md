## ⚙️ Cara Pakai (Contoh: Audio Cutter)

1. Pastikan sudah install:

   ```bash
   sudo pacman -S ffmpeg
   ```

2. Edit bagian ini sesuai kebutuhan:

   ```bash
   INPUT="path/ke/lagu.mp3"
   OUTPUT_DIR="path/output"
   ```

3. Jalankan:

   ```bash
   chmod +x script.sh
   ./script.sh
   ```
